# Data Dictionary

## umme_sites_messenia

| Column | Description |
|-------|-------------|
site_no | Unique site identifier
location_desc | Textual description of location
physical_desc | Original geomorphological description
region | Administrative region
geomorphology_class | Derived geomorphology class
topographic_zone | Derived topographic classification


## umme_archaelogical_data

| Column | Description |
|-------|-------------|
site_no | Site identifier
bibliography | Bibliographic references
pottery_analysis | Chronological indicators
extent_desc | Text description of site size
area_ha | Extracted site area in hectares
settlement_or_cemetery | Site classification
